Что делает инструмент

Загружает документ в базу данных для поиска по смыслу — после загрузки документ можно искать через агента по документам, задавая вопросы по содержимому.

Принимаются .pdf, .doc, .docx, .xls, .xlsx, .ppt, .pptx, .txt, .rtf, .odt, изображения. Максимум — 50 МБ.

Обрабатываю документ: OCR → нарезка → эмбеддинг → сохранение
