Что делает инструмент

Загрузите несколько Excel-файлов (.xlsx, .xlsm) с таблицами. Инструмент объединит все листы из всех файлов в одну таблицу — заголовки склеит по смыслу, строки сложит подряд.

Перетягивайте файлы в списке для изменения порядка. Доступны: удаление дубликатов, сортировка, скрытие столбцов и превью результата перед скачиванием.
