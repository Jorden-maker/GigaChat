    Тест без LLM (прямые action-вызовы)

    Workflow поддерживает direct-режим: если в теле есть action, LLM пропускается. Тут — UI поверх этого режима. Нужны заполненные Plane-настройки и валидный planner-токен.

    1. Список проектов

    action=list_projects, params={}

    2. Список задач в проекте

    action=list_issues, params={ project_name }

    3. Создать задачу

    action=create_issue, params={ project_name, name, priority? }

    4. Обновить задачу

    action=update_issue, params={ project_name, issue_name, priority?, name? }

    5. Удалить задачу

    action=delete_issue, params={ project_name, issue_name }

    6. Поиск задач

    action=search_issues, params={ project_name, query }

    7. Получить задачу (детали)

    action=get_issue, params={ project_name, issue_name }

    8. Изменить статус

    action=change_status, params={ project_name, issue_name, state }

    9. Установить срок (target_date)

    action=set_deadline, params={ project_name, issue_name, target_date: 'YYYY-MM-DD' }

    10. Изменить приоритет

    action=update_issue, params={ project_name, issue_name, priority }

    11. Добавить комментарий

    action=add_comment, params={ project_name, issue_name, comment }
